# AdminAFFriend

O **AdminAFFriend** é um painel administrativo premium desenvolvido para gerir de forma escalável todos os recursos da plataforma **AFFriend**.  
Ele oferece uma experiência **intuitiva, responsiva e centrada no usuário**, permitindo que administradores tenham controle total sobre cursos, lições, produtos, usuários e notificações.

## 🚀 Funcionalidades

- **Gestão de Cursos** — criação, edição, progresso e avaliações.  
- **Gestão de Lições** — administração modular e acompanhamento detalhado.  
- **Gestão de Produtos** — cadastro, estoque e integração com o Marketplace.  
- **Gestão de Usuários** — perfis, permissões e papéis (admin, criador, estudante).  
- **Notificações** — envio e monitoramento de alertas internos.  
- **UI/UX Premium** — paleta AFFriend (branco, vermelho, preto), responsivo para desktop e mobile.  
- **Integração com Firebase** — segurança e persistência de dados.  
- **Automação de processos** — sitemaps, APIs e regras de acesso.
