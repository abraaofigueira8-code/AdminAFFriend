AFFriend Admin — Documentação Oficial (Dev‑Friendly)

1. Introdução

O AFFriend Admin é o painel administrativo oficial do ecossistema AFFriend. Ele foi projetado para ser:

  • Rápido
  • Modular
  • Fácil de manter
  • Intuitivo para novos devs
  • Escalável para novos módulos

O objetivo é permitir que administradores, gestores e desenvolvedores controlem todo o sistema de forma simples e eficiente.

2. Instalação e Configuração

Requisitos mínimos

Servidor

  • PHP 8+ ou Node 18+ (dependendo do backend)
  • MySQL/MariaDB ou PostgreSQL
  • Nginx ou Apache

Frontend

  • HTML5
  • CSS3
  • JavaScript ES6
  • Bootstrap 5 (ou framework equivalente usado no projeto)

Instalação

  1. Clone o repositório: 
    git clone https://github.com/affriend/admin.git
  2. Instale dependências (se houver): 
    npm install
  3. Configure o arquivo .env:
    • URL base
    • Credenciais do banco
    • Chaves de API
  4. Inicie o servidor: 
    npm run dev

    ou

    php artisan serve

Primeiro login

  • Acesse: /index.html
  • Usuário padrão: admin@affriend.com
  • Senha padrão: admin123

4. Autenticação e Segurança

Fluxo de login

  1. Usuário envia email + senha
  2. Backend valida credenciais
  3. Token de sessão é criado
  4. Usuário é redirecionado ao dashboard

Proteção de rotas

  • Todas as páginas dentro de /pages exigem sessão ativa
  • Middleware verifica token antes de renderizar conteúdo

Recuperação de senha

  • Envio de email com link temporário
  • Token expira em 15 minutos

10. FAQ

“O login não funciona”
→ Verifique credenciais e conexão com o banco.

“Sidebar não aparece”
→ Confirme se o include está correto no HTML.

“Gráficos não carregam”
→ Verifique se o Chart.js/ApexCharts está importado.
